# Proyecto Tienda Virtual

Este proyecto es una aplicación web desarrollada con versiones en **Angular** y **React**, utilizando **Firebase** como backend y gestor de base de datos. A continuación, se describen las especificaciones técnicas y pasos para ejecutar cada versión.

---

## **Tecnologías Utilizadas**
- **Backend y Base de Datos:** [Firebase](https://firebase.google.com)
  - Autenticación de usuarios con Firebase Authentication.
  - Gestión de datos mediante Firebase Realtime Database o Firestore.

---

## **Autenticación de Usuarios**
Para iniciar sesión en la aplicación, utiliza las siguientes credenciales válidas:

- **Email:** `usuario.tienda@mail.com`  
- **Password:** `123456`
