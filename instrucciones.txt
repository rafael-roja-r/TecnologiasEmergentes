* Como Backend y gestor de BD se utilizó Firebase.
    * Así también para la autenticación de usuario (Login) se utilizó Firebase.
    * Para validar el login las credenciales validas son:
          # Email: usuario.tienda@mail.com
          # Password: 123456
    * Para la versión con Angular antes de ejecutar el comando "ng serve" del CLI se debe ejecutar el comando "npm install" para instalar la dependencias necesarias (que no fueron incluidas en la carpeta del proyecto a modo de minimizar el peso del archivo .zip).
    * Para la versión con React las dependencias están incluidas en la carpeta del proyecto, para correr la aplicación se utiliza el comando "npm start".
